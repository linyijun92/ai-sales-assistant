启动一个多 Agent 团队来开发"AI 助销助手"项目。

## 项目背景
- 原有系统：E 销系统（CRM 系统）
- 目标：利用 LLM 能力将 CRM 增强为 AI Agent 应用

## 核心功能
1. 销售顾问通过语音或文字操控 E 销系统功能（客户建档、添加跟进记录等）
2. AI Agent 以个人助理形式整理客户信息、试驾情况报告等

## 团队配置

### 1. 产品经理 Agent
- 职责：需求拆解、场景定义、验收标准
- 需要深入理解销售顾问的工作流程
- 输出：需求文档、用户故事、验收标准

### 2. 系统架构师 Agent
- 职责：接口定义、技术选型、Prompt 工程设计
- 需要考虑移动端特性、语音识别集成、CRM 系统对接
- 输出：架构设计文档、API 规范、Prompt 模板

### 3. 全栈开发 Agent
- 职责：全栈代码开发
- 需要开发移动端应用（建议 Flutter）、后端 API、LLM 集成
- 输出：完整的代码实现

### 4. 测试与 QA Agent
- 职责：模拟用户测试、数据校验、Bug 报告
- 需要模拟销售顾问的真实使用场景
- 输出：测试报告、Bug 列表、验收结果

## 工作流程
1. 产品经理 Agent → 需求分析与场景定义（创建 docs/requirements.md）
2. 系统架构师 Agent → 技术方案设计（创建 docs/architecture.md）
3. 全栈开发 Agent → 代码开发（创建 src/ 目录和代码）
4. 测试与 QA Agent → 测试与质量保证（创建 tests/ 目录和测试用例）
